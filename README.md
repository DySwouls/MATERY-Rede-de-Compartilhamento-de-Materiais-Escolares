# MATERY-Rede-de-Compartilhamento-de-Materiais-Escolares
MATERY é uma plataforma web que conecta estudantes que desejam doar materiais escolares a alunos que necessitam desses recursos. O projeto promove reutilização, colaboração e acesso à educação por meio do compartilhamento de livros, apostilas e materiais acadêmicos, alinhando-se à ODS 17.
